BuyItem

Buy an item to a user. GuildID is not needed for global use.

Source:
  • lib/eco.js, line 517
Example
await client.ecoBuyItem({ GuildID: "881789379553656872", Item: "very expensive pancakes" | "0", user: "753974636508741673" })
Returns:

return ITEM_NOT_FOUND or NOT_ENOUGH_CASH or ALREADY_PURCHASED or array of item objects

Type
string | Array
